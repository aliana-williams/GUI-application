
## Overview
This is a GUI program created in Python with Tkinter that allows users to perform multiple calculations and conversions in one application.

## Features
- Adds two numbers together
- Converts Fahrenheit to Celsius
- Interactive graphical user interface
- Clear/reset function

## Technologies Used
- Python
- Tkinter


## What I Learned
This project helped me practice building interactive applications with Tkinter, handling user input, organizing multiple functions in one program, and displaying outputs  in GUI.
